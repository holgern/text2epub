# Introduction

This small book collects three essays about turning plain text into EPUB.

It is generated entirely with `text2epub` from a folder of Markdown files.

## What you will see

- Multiple chapters sorted into spine order.
- Headings with stable identifiers.
- A local image, a table, and a code block.
